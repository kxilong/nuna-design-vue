## 特性

-   提炼自企业级中后台产品的交互语言和视觉风格。
-   开箱即用的高质量 Vue 组件。

## 支持环境

-   现代浏览器。1.x 版本支持 IE 9+（需要 [polyfills](https://www.antdv.com/docs/vue/getting-started-cn/#兼容性)）
-   支持 Vue 2 和 Vue 3
